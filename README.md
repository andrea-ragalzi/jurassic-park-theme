## Installation

1. Open **Extensions** in VS Code (`Ctrl+Shift+X` / `Cmd+Shift+X`)
2. Search for **"Jurassic Park Theme"**
3. Click **Install**
4. Open Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
5. Type **"Color Theme"** and select **"Jurassic Park Theme"**
