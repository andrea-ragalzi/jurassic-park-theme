# Change Log

All notable changes to the "jurassic-park-theme" extension will be documented in this file.

## [1.0.3] - 2025-12-10

### Added

- **Semantic Highlighting**: Enabled and configured semantic token colors for better distinction between variables, constants, parameters, and properties across languages.
- **Markdown Support**: Added specific styling for Markdown headings using the iconic Amber color.
- **Theme Icon**: Added a dedicated icon for the extension.

### Changed

- **Color Palette Overhaul**: Completely optimized the palette to match the original _Jurassic Park (1993)_ movie aesthetics:
  - **Background**: Deep Jungle Night (`#121612`)
  - **Keywords**: Amber Gold (`#D4AF37`)
  - **Strings**: Lush Vegetation Green (`#87b358`)
  - **Types/Classes**: T-Rex Red (`#ec1b31`)
  - **Constants**: Warm Orange (`#ff8c42`)
- **Accessibility**:
  - Improved contrast for remote status bar items.
  - Increased opacity for editor selection for better visibility.
- **JSX/TSX**: Neutralized color for JSX/TSX children to improve readability of component structures.
- **Internal**: Translated all code comments to English for better maintainability.

## [1.0.0] - 2025-12-04

- Initial release of the Jurassic Park Theme.
